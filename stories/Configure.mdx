# Welcome to RoomReimagine AI Storybook

This is the component library for the RoomReimagine AI application.

## Available Components

Browse the sidebar to explore our components:

- **Example/Button** - Basic button component with various states
- **Example/Header** - Application header with navigation
- **Example/Page** - Page layout template
- **System/Design Tokens** - Color palette and design system documentation

## Getting Started

Each component has interactive controls in the "Controls" panel below. 
You can modify props and see changes in real-time.

## About RoomReimagine AI

RoomReimagine AI is a full-stack interior design application that leverages Google's Gemini Vision API to intelligently redesign room interiors while preserving specific user-selected objects.

### Key Features:
- Upload room images
- Specify elements to keep unchanged
- Apply various design styles
- Configurable creativity levels
